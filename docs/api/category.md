```text
GET {{host}}/category?page={{page}}
```

```json
{
    "code": 200,
    "msg": "OK",
    "data": [
        {
            "id": 1,
            "name": "大一课程"
        },
        {
            "id": 2,
            "name": "基础课程"

```

```text
GET {{host}}/category/s?q={{search_query}}&page={{page}}
```
