# 选课宝典

选课宝典是一个以评论社区为特色的，提供交大所有课程、授课老师介绍以及部分课程资料（包括课程ppt、学霸笔记、试题卷）的选课辅助工具

<https://course.eeyes.net>

后端提供纯API接口，前端为微信小程序
